# Car OUT Jam Puzzle Game - Development & Bug Fix Guide

This guide provides the complete solution to fix the **rapid clicking tackle bug** and details how to implement a **premium, interactive Spin Wheel Game** inside your Unity project. 

---

## Part 1: Curing the Rapid Click "Tackle" Bug

### The Cause of the Bug
In `Assets/Script/Carout.cs`, the `DoTackle()` method retrieves the car's *current* world position at the exact frame the method is invoked, using `Vector3 originalPosition = transform.position;`. 
If a player click-spams the car while it is still in the middle of a tackle animation (tacking forward or returning back), `originalPosition` is set to this intermediate animated position. The new tween then returns the car to this intermediate spot, causing the car to drift off the grid, fly out of alignment, and crash the level layout.

### The Solution
1. **Prevent Overlapping Tweens**: Add a boolean state flag `isTackling` to track whether a tackle animation is currently running. If it is active, ignore subsequent clicks.
2. **Expose `IsTackling`**: Allow `CarController.cs` to query this state so that sound effects, camera shakes, and logic don't spam.
3. **Lock to Grid Center**: Fallback to caching the slot's actual world position from `SpawnCars.Instance` if possible, guaranteeing that the car always returns to the exact center of its slot.

Open `Assets/Script/Carout.cs` and modify the class to prevent concurrent tackles:

```csharp
public class Carout : MonoBehaviour
{
    public float rayDis = 5f;
    // public Transform Position;

    public float dashSpeed = 0.2f;
    public float returnSpeed = 0.5f;

    // public CarMover carMover;

    // public GameObject ColliderTrigger;
    public bool isBlocked { get; private set; }

    public Vector2Int currentGridIndex; 
    [Header("Dynamic Tackle Settings")]
    public float tackleCollsionOffset = 4f;
    private Vector2Int blockingGridIndex;

    [SerializeField] private ParticleSystem impactParticle;

    // Tackle state tracking
    private bool isTackling = false;
    public bool IsTackling => isTackling;

    private void OnEnable()
    {
        isTackling = false; // Reset state when recycled or enabled
    }

    private void OnDisable()
    {
        isTackling = false;
    }

    public bool CheckForBlockage()
    {
        if (SpawnCars.Instance == null) return false;

        // 1. Check for block using the method inside SpawnCars
        isBlocked = SpawnCars.Instance.CheckBlockageForCar(currentGridIndex, transform.eulerAngles, out Vector2Int targetGridIndex);

        // 2. If it is NOT blocked, make the slot isOccupied false simply
        if (!isBlocked)
        {
            SpawnCars.Instance.SetSlotOccupation(currentGridIndex.x, currentGridIndex.y, false);
            currentGridIndex = targetGridIndex; // Car tracks its new position
        }
        else
        {
            blockingGridIndex = targetGridIndex;
        }

        return isBlocked;
    }

    public void DoTackle()
    {
        if (isTackling) return; // Ignore overlap if already tackling
        isTackling = true;

        Vector3 originalPosition = transform.position;

        // Cache the exact center of the slot to prevent any drift
        if (SpawnCars.Instance != null)
        {
            var slot = SpawnCars.Instance.GetSlotAt(currentGridIndex.x, currentGridIndex.y);
            if (slot != null)
            {
                originalPosition = slot.WorldPosition;
            }
        }

        Vector3 tacklePosition = originalPosition + (transform.forward * 1.5f); // Fallback nudge

        if (SpawnCars.Instance != null && isBlocked)
        {
            // Retrieve the slot data for the blocking car
            var blockingSlot = SpawnCars.Instance.GetSlotAt(blockingGridIndex.x, blockingGridIndex.y);
            if (blockingSlot != null)
            {
                Vector3 blockingWorldPos = blockingSlot.WorldPosition;
                Vector3 dir = (blockingWorldPos - originalPosition).normalized;

                // Distance in world units between the two cars
                float distance = Vector3.Distance(originalPosition, blockingWorldPos);

                // If they are far enough apart, slide right up to the obstacle with a collision offset
                if (distance > tackleCollsionOffset)
                {
                    tacklePosition = blockingWorldPos - dir * tackleCollsionOffset;
                }
                else
                {
                    // If they are already touching or extremely close, do a small proportional nudge
                    tacklePosition = originalPosition + (dir * (distance * 0.4f));
                }
            }
        }

        // Create the DOTween sequence
        Sequence tackleSequence = DOTween.Sequence();

        // 1. Move forward fast to the calculated tackle position
        tackleSequence.Append(transform.DOMove(tacklePosition, dashSpeed).SetEase(Ease.OutFlash));

        // Play particle EXACTLY on impact
        tackleSequence.AppendCallback(() =>
        {
            if (impactParticle != null)
            {
                impactParticle.Play();
            }
        });

        // 2. Small shake to add physical impact
        tackleSequence.Append(transform.DOShakeRotation(0.2f, 10f));

        // 3. Move back to original position
        tackleSequence.Append(transform.DOMove(originalPosition, returnSpeed).SetEase(Ease.InQuad));

        // Reset tackle state on complete
        tackleSequence.OnComplete(() =>
        {
            isTackling = false;
        });
    }
}

## Part 2: The Premium Spin Wheel Mini-Game

We will build a simple, responsive, and beautiful **Spin Wheel Panel** with betting options. It consumes coins (10, 20, 50, or 100 coins) and scales the rewards proportionally.

### Sector Distribution (8 Sectors)
* **Sector 0 (0°)**: 20 Coins
* **Sector 1 (45°)**: +1 Fill Slot Power-up
* **Sector 2 (90°)**: 50 Coins
* **Sector 3 (135°)**: +1 Rearrange Queue Power-up
* **Sector 4 (180°)**: 100 Coins
* **Sector 5 (225°)**: +1 Helicopter VIP Power-up
* **Sector 6 (270°)**: 200 Coins *(JACKPOT!)*
* **Sector 7 (315°)**: +2 Rearrange Queue Power-ups

### How Betting Works
Players can increase the Bet multiplier: `x1 (10 Coins)`, `x2 (20 Coins)`, `x5 (50 Coins)`, or `x10 (100 Coins)`.
* **Coin wins** are multiplied directly (e.g. at x5, winning 50 Coins adds **250 Coins**!).
* **Power-up wins** are also multiplied (e.g. at x2, winning +1 Helicopter gives **+2 Helicopter VIPs**!).

### Step 3: Spin Wheel C# Script (`SpinWheelController.cs`)
Create a new file `Assets/Script/AdditionalFeatures/SpinWheelController.cs` and paste the following implementation:

```csharp
using System;
using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.UI;
using TMPro;
using DG.Tweening;

public class SpinWheelController : MonoBehaviour
{
    public enum RewardType
    {
        Coins,
        FillSlotPowerup,
        RearrangePowerup,
        HelicopterPowerup
    }

    [System.Serializable]
    public class WheelSector
    {
        public string rewardName;
        public RewardType rewardType;
        public int baseRewardAmount;
        public Sprite rewardIcon;
    }

    [Header("UI Canvas Elements")]
    [SerializeField] private GameObject spinPanel;
    [SerializeField] private Transform wheelTransform;
    [SerializeField] private Transform pointerArrow;
    [SerializeField] private Button spinButton;
    [SerializeField] private TextMeshProUGUI spinCostText;
    [SerializeField] private TextMeshProUGUI playerCoinsText;
    
    [Header("Bet System")]
    [SerializeField] private TextMeshProUGUI betMultiplierText;
    [SerializeField] private Button increaseBetBtn;
    [SerializeField] private Button decreaseBetBtn;
    [SerializeField] private int baseSpinCost = 10;
    [SerializeField] private List<int> betMultipliers = new List<int> { 1, 2, 5, 10 };
    private int currentBetIndex = 0;

    [Header("Win Popup Panel")]
    [SerializeField] private GameObject winPopupPanel;
    [SerializeField] private TextMeshProUGUI winTitleText;
    [SerializeField] private TextMeshProUGUI winRewardText;
    [SerializeField] private Image winRewardIconImage;
    [SerializeField] private Button winCloseButton;

    [Header("Aesthetic Settings")]
    [SerializeField] private float spinDuration = 4f;
    [SerializeField] private Ease spinEase = Ease.OutQuad;
    [SerializeField] private float pointerPunchAngle = -18f;

    [Header("Wheel Configuration (8 Sectors)")]
    public List<WheelSector> sectors = new List<WheelSector>()
    {
        new WheelSector { rewardName = "20 Coins", rewardType = RewardType.Coins, baseRewardAmount = 20 },
        new WheelSector { rewardName = "Fill Slot", rewardType = RewardType.FillSlotPowerup, baseRewardAmount = 1 },
        new WheelSector { rewardName = "50 Coins", rewardType = RewardType.Coins, baseRewardAmount = 50 },
        new WheelSector { rewardName = "Rearrange Queue", rewardType = RewardType.RearrangePowerup, baseRewardAmount = 1 },
        new WheelSector { rewardName = "100 Coins", rewardType = RewardType.Coins, baseRewardAmount = 100 },
        new WheelSector { rewardName = "Helicopter VIP", rewardType = RewardType.HelicopterPowerup, baseRewardAmount = 1 },
        new WheelSector { rewardName = "200 Coins", rewardType = RewardType.Coins, baseRewardAmount = 200 },
        new WheelSector { rewardName = "Rearrange x2", rewardType = RewardType.RearrangePowerup, baseRewardAmount = 2 }
    };

    private bool isSpinning = false;
    private int CurrentMultiplier => betMultipliers[currentBetIndex];
    private int CurrentSpinCost => baseSpinCost * CurrentMultiplier;

    private void Start()
    {
        // Setup initial UI states
        UpdateBetUI();
        UpdatePlayerCoinsUI();
        
        if (winPopupPanel != null) winPopupPanel.SetActive(false);

        // Add Listeners
        spinButton.onClick.AddListener(OnSpinButtonClicked);
        increaseBetBtn.onClick.AddListener(IncreaseBet);
        decreaseBetBtn.onClick.AddListener(DecreaseBet);
        if (winCloseButton != null) winCloseButton.onClick.AddListener(CloseWinPopup);

        // Listen for save updates to keep coin display accurate
        SaveData.OnDataChanged += UpdatePlayerCoinsUI;
    }

    private void OnDestroy()
    {
        SaveData.OnDataChanged -= UpdatePlayerCoinsUI;
    }

    public void OpenSpinWheel()
    {
        spinPanel.SetActive(true);
        UpdatePlayerCoinsUI();
        UpdateBetUI();
    }

    public void CloseSpinWheel()
    {
        if (isSpinning) return; // Prevent closing mid-spin
        spinPanel.SetActive(false);
    }

    private void UpdateBetUI()
    {
        if (betMultiplierText != null)
            betMultiplierText.text = $"x{CurrentMultiplier}";

        if (spinCostText != null)
            spinCostText.text = $"{CurrentSpinCost}";

        // Disable increase/decrease buttons at limits
        increaseBetBtn.interactable = currentBetIndex < betMultipliers.Count - 1 && !isSpinning;
        decreaseBetBtn.interactable = currentBetIndex > 0 && !isSpinning;
    }

    private void UpdatePlayerCoinsUI()
    {
        if (playerCoinsText != null && SaveData.Instance != null)
        {
            playerCoinsText.text = SaveData.Instance.CurrentSave.currency.ToString();
        }
    }

    private void IncreaseBet()
    {
        if (isSpinning) return;
        if (currentBetIndex < betMultipliers.Count - 1)
        {
            currentBetIndex++;
            UpdateBetUI();
            SoundManager.Instance.PlaySound(SoundManager.SoundName.Click);
        }
    }

    private void DecreaseBet()
    {
        if (isSpinning) return;
        if (currentBetIndex > 0)
        {
            currentBetIndex--;
            UpdateBetUI();
            SoundManager.Instance.PlaySound(SoundManager.SoundName.Click);
        }
    }

    private void OnSpinButtonClicked()
    {
        if (isSpinning) return;

        // Check if user has enough coins
        if (SaveData.Instance == null) return;
        int currentCoins = SaveData.Instance.CurrentSave.currency;

        if (currentCoins < CurrentSpinCost)
        {
            // Play a buzz/deny sound or flash text red
            Debug.LogWarning("Not enough coins to spin!");
            spinCostText.transform.DOShakePosition(0.5f, 5f, 20);
            return;
        }

        // Deduct Coins
        SaveData.Instance.AddCurrency(-CurrentSpinCost);
        SoundManager.Instance.PlaySound(SoundManager.SoundName.Click);
        StartSpinning();
    }

    private void StartSpinning()
    {
        isSpinning = true;
        spinButton.interactable = false;
        increaseBetBtn.interactable = false;
        decreaseBetBtn.interactable = false;

        // 1. Choose winning sector randomly
        int wonSectorIndex = UnityEngine.Random.Range(0, sectors.Count);

        // 2. Calculate target rotation angle on Z
        // 8 sectors = 45 degrees each. Sector 0 is centered at 0 degrees.
        // Clockwise layout requires rotating counter-clockwise to shift clockwise elements under the pointer:
        // Angle to rotate = 360 - (wonSectorIndex * 45)
        float sectorAngle = 360f - (wonSectorIndex * 45f);
        float extraRotations = 360f * 6f; // Spin 6 full loops for anticipation
        float targetZAngle = sectorAngle + extraRotations;

        // Reset wheel rotation first to avoid accumulation bugs
        wheelTransform.localEulerAngles = Vector3.zero;

        // 3. Pointer Jiggle simulation during rotation
        // Jiggle pointer arrow rotation dynamically when sectors pass by
        Sequence pointerSequence = DOTween.Sequence();
        float jiggleInterval = spinDuration / 30f; // Jiggle 30 times during the spin
        for (int i = 0; i < 26; i++)
        {
            float delay = (float)i * (spinDuration / 28f);
            pointerSequence.InsertCallback(delay, () =>
            {
                if (pointerArrow != null)
                {
                    pointerArrow.DOPunchRotation(new Vector3(0, 0, pointerPunchAngle * (1f - (delay / spinDuration))), 0.15f, 2, 0.5f);
                }
            });
        }

        // 4. Spin the wheel transform using DOTween
        wheelTransform.DORotate(new Vector3(0, 0, targetZAngle), spinDuration, RotateMode.FastBeyond360)
            .SetEase(spinEase)
            .OnComplete(() =>
            {
                // Final pointer peg snap sound/animation
                if (pointerArrow != null)
                {
                    pointerArrow.DOPunchRotation(new Vector3(0, 0, pointerPunchAngle * 0.8f), 0.3f, 4, 1);
                }
                
                // Process won sector
                AwardPlayerPrize(wonSectorIndex);
            });
    }

    private void AwardPlayerPrize(int sectorIndex)
    {
        isSpinning = false;
        spinButton.interactable = true;
        UpdateBetUI();

        WheelSector wonSector = sectors[sectorIndex];
        int finalRewardAmount = wonSector.baseRewardAmount * CurrentMultiplier;

        // Apply reward to SaveData
        switch (wonSector.rewardType)
        {
            case RewardType.Coins:
                SaveData.Instance.AddCurrency(finalRewardAmount);
                break;
                
            case RewardType.FillSlotPowerup:
                int currentFill = SaveData.Instance.CurrentSave.fillSlotCount;
                SaveData.Instance.SetFillSlotCount(currentFill + finalRewardAmount);
                break;
                
            case RewardType.RearrangePowerup:
                int currentRearrange = SaveData.Instance.CurrentSave.rearrangeCount;
                SaveData.Instance.SetRearrangeCount(currentRearrange + finalRewardAmount);
                break;
                
            case RewardType.HelicopterPowerup:
                int currentHeli = SaveData.Instance.CurrentSave.helicopterCount;
                SaveData.Instance.SetHelicopterCount(currentHeli + finalRewardAmount);
                break;
        }

        // Save progress explicitly (SaveData methods call Save automatically, but just to be sure)
        SaveData.Instance.Save();

        // Refresh UI layers
        if (PowerUps.Instance != null)
        {
            PowerUps.Instance.LoadFromSave();
        }

        // Trigger Win Popup Overlay
        ShowWinPopup(wonSector, finalRewardAmount);
    }

    private void ShowWinPopup(WheelSector sector, int amount)
    {
        if (winPopupPanel == null) return;

        SoundManager.Instance.PlaySound(SoundManager.SoundName.carFull); // Premium winning sound

        winPopupPanel.SetActive(true);
        winPopupPanel.transform.localScale = Vector3.zero;
        winPopupPanel.transform.DOScale(1f, 0.4f).SetEase(Ease.OutBack);

        if (winTitleText != null)
        {
            winTitleText.text = sector.rewardType == RewardType.Coins ? "COINS WON!" : "POWER-UP WON!";
        }

        if (winRewardText != null)
        {
            winRewardText.text = sector.rewardType == RewardType.Coins ? $"+{amount} Coins" : $"+{amount} {sector.rewardName}";
        }

        if (winRewardIconImage != null && sector.rewardIcon != null)
        {
            winRewardIconImage.sprite = sector.rewardIcon;
            winRewardIconImage.gameObject.SetActive(true);
        }
        else if (winRewardIconImage != null)
        {
            winRewardIconImage.gameObject.SetActive(false); // Hide image if no custom sprite assigned
        }
    }

    private void CloseWinPopup()
    {
        SoundManager.Instance.PlaySound(SoundManager.SoundName.Click);
        winPopupPanel.transform.DOScale(0f, 0.2f).SetEase(Ease.InBack).OnComplete(() =>
        {
            winPopupPanel.SetActive(false);
        });
    }
}

## Part 3: Unity Scene Integration Step-by-Step Setup

Follow these exact steps in the Unity Editor to construct the Wheel interface and wire up the logic:

### Step 1: UI Structure Hierarchy
Create the following layout under your main UI Canvas (`Canvas`):

- **Canvas**
  - **OverlayBackground**: Semi-Transparent Dark Image (locks gameplay focus)
    - **SpinWheelPanel**: Main UI Panel
      - **CloseButton**: Button (closes panel)
      - **Header Text**: "SPIN WHEEL"
      - **WheelFrame**: Decorative Outer Circle Frame Image
        - **TheWheel**: Image wedged with alternating sector slices
        - **Pointer**: Needle image at the top pointing downwards
      - **CoinsBar**: Displays player's current coin balance (bind to save data)
      - **BetMultiplierPanel**: Horizontal group to modify betting cost
        - **DecreaseBetBtn**: Button "-"
        - **BetMultiplierText**: TMPro Text (e.g. "x1")
        - **IncreaseBetBtn**: Button "+"
      - **SpinButton**: Main execution button
        - **CostText**: TMPro Text (e.g. "10")
      - **WinPopupPanel**: Block Panel for showing wins
        - **WinTitleText**: TMPro Text
        - **WinRewardText**: TMPro Text
        - **WinRewardIconImage**: Image
        - **CollectButton**: Button (closes popup)

### Step 2: Creating the Wheel Circle Graphic
For a gorgeous look without complex textures, you can create a circular segmented layout directly in Photoshop/Figma or use a basic circle:
1. Slice a circle into **8 equal wedges (45 degrees each)**. Use alternating colors (like vibrant violet `#8E44AD` and deep gold `#F1C40F`).
2. Label or add small icons near the rim of each segment to represent the rewards (20 Coins, +1 Fill, 50 Coins, +1 Rearrange, 100 Coins, +1 Helicopter, 200 Coins, +2 Rearrange).
3. Export it as a PNG sprite. Import into Unity as a **Sprite (2D and UI)**.

### Step 3: Wire Up the `SpinWheelController` Component
1. Create an empty GameObject inside the UI Canvas named `SpinWheelSystem` and drag the `SpinWheelController.cs` script onto it.
2. Drag and drop the corresponding references into the inspector fields:
   * **Spin Panel**: The `SpinWheelPanel` GameObject.
   * **Wheel Transform**: The `TheWheel` RectTransform (the circle child image that will rotate).
   * **Pointer Arrow**: The `Pointer` RectTransform (at 12 o'clock).
   * **Spin Button**: The main button in the bottom-middle.
   * **Spin Cost Text**: The TextMeshPro component showing the cost on the button (e.g. `10`).
   * **Player Coins Text**: The text displaying the current total coins (e.g. `1000`).
   * **Bet Multiplier Text**: The text showing `x1`, `x2`, etc.
   * **Increase Bet Btn**: The `+` button.
   * **Decrease Bet Btn**: The `-` button.
   * **Win Popup Panel**: The popup overlay panel.
   * **Win Title Text**: The header in the popup.
   * **Win Reward Text**: The main reward notification.
   * **Win Close Button**: The "Collect" button in the win popup.
3. Assign beautiful sprites to the `sectors` icons in the Inspector if you want to display the visual icons in the Win Popup.

### Step 4: Connecting the Spin Wheel to Game UI
1. Create a "Spin Wheel" button somewhere in your Main Menu or Game View UI.
2. Hook up its `OnClick()` event to call: `SpinWheelSystem -> SpinWheelController.OpenSpinWheel()`.
3. Set the `SpinWheelPanel` GameObject to be disabled (`SetActive(false)`) by default on startup.
